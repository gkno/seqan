This is a SeqAn snapshot.
See www.seqan.de

The following compilers are supported:
for Linux: gcc 3.0, 3.2, 3.3, 3.4, 4.1
for Windows: visual c++ 2003, 2005

Installation for g++:
- rename platform_linux_g++.h to platform.h
- copy the file to folder seqan, replace the old platform.h there
- add the seqan installation directory (that one containing "seqan", "demos" and "doc") to the include search path
